# GigaChatCLI
AI-agent based on GigaChat model.

## Installation
1. `git clone https://github.com/RussianCLI/GigaChatCLI && cd GigaChatCLI`
2. `python3 -m venv venv && source venv/bin/activate`
3. `pip install -r requirements.txt`
4. `cp .env.example .env`
5. Paste your GigaChat credentials into .env
6. Run! `python3 src/main.py`
